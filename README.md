# Flutter MVVM Sensor App
#### -  This project made for a sensors company task. The app is making POST request and getting data from response.
 
 ### The app is coding with:
 
 #### -MVVM Design Pattern
 #### -Dio
 #### -Json Serializable
 #### -Provider State Management
 #### -Shared Preferences
 #### -Lottie
